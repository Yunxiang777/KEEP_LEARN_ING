1️⃣ Two Sum

給定一個整數陣列 nums 和一個目標值 target，請找出陣列中兩個數，使它們的和等於目標值。

  • 你可以假設每組輸入只有一個答案。
  • 同一個元素不能重複使用。
輸入: nums = [2,7,11,15], target = 9
輸出: [0,1]

解答

public int[] TwoSum(int[] nums, int target) {
    int len = nums.Length;
    for (int i = 0; i < len - 1; i++) {
        for (int j = i + 1; j < len; j++) {
            if (nums[i] + nums[j] == target)
                return new int[] { i, j };
        }
    }
    return null;
}

此方法為暴力解法 (Brute Force)，時間複雜度 O(n²)。

2️⃣ Reverse String

請反轉給定的字元陣列。必須原地修改輸入陣列，並使用 O(1) 額外空間。

輸入: ['h','e','l','l','o']
輸出: ['o','l','l','e','h']

解答

public class Solution {
    public void ReverseString(char[] s) {
        int left = 0, right = s.Length - 1;
        while (left < right) {
            char temp = s[left];
            s[left] = s[right];
            s[right] = temp;
            left++;
            right--;
        }
    }
}

雙指針 (Two Pointers) 技巧，時間 O(n)，空間 O(1)。
